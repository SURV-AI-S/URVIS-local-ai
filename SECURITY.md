# Security Policy

URVIS Local AI is a local-first static web interface. It does not provide authentication.

## Security guidance

- Keep Ollama local.
- Do not expose port `11434` publicly.
- Do not paste secrets, passwords, API keys, private keys, or sensitive company code.
- Review any model license and privacy implications before use.
- Treat exported chat JSON files as potentially sensitive.

## Reporting issues

Report security issues through GitHub Issues unless the repository owner provides a private reporting channel.
