# Usage

URVIS Local AI works best with pasted code, pasted errors, and direct questions.

## Pasted code

Paste a Unity C# script into the composer and ask for an explanation, risk review, or small improvement.

## Unity errors

Paste the full Unity Console error and any relevant script lines. The assistant can help explain likely causes and safe fixes.

## Refactors

Paste a small snippet and ask for a safe refactor. For Unity projects, avoid unnecessary renaming of public or `[SerializeField]` fields because Inspector references may break.

## Script generation

Ask for a small Unity `MonoBehaviour` and describe the behavior clearly. Review generated code before adding it to a project.

## Important limitation

The app does not automatically read folders, scan project files, index codebases, or edit files. Paste the context you want the model to use.
