# Contributing

Thanks for helping improve URVIS Local AI.

## Principles

- Keep the app local-first.
- Do not add model weights, downloaded models, Ollama binaries, or IPEX-LLM binaries.
- Do not add fake file-reading, automatic project scanning, or codebase indexing features.
- Keep the UI honest about limitations.
- Prefer vanilla HTML, CSS, and JavaScript unless there is a strong reason to add a dependency.
- Keep changes small, readable, and easy to review.

## Before submitting changes

- Run the app locally in a browser.
- Check that chat, settings, model selection, and prompt templates still work.
- Make sure no private paths, logs, secrets, screenshots, model files, or cache folders are included.
- Keep README and docs updated when behavior changes.
